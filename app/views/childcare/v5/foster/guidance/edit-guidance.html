Guidance on changing foster family details

Change carer or partner details

Use the ‘Change’ links to correct any mistakes, or to update details if the carer’s partner changes.

Making changes after an eligibility code has been issued

After an eligibility code has been issued, you can only make acceptable changes to partner and carer details. For example, you can:

  • correct an error in a carer or partner National Insurance number or date of birth
  • correct a spelling mistake in a carer or partner name
  • change the name of a carer or partner from a legal name to a preferred name, or the other way round
  • change the name of the carer when they legally change their name
  • add or remove a partner
  • change the partner to a different person
  • change the postcode if the carer has moved

These acceptable changes will be applied when an eligibility code is next issued or reconfirmed and a new eligibility event is created. These changes will not be retrospectively applied to past eligibility events. If there is a mistake in an issued eligibility event and it needs correcting, contact the service desk.

Changes that are not acceptable after an eligibility code has been issued

There are some changes to personal details that local authorities must not make after a code has been issued. These changes will create problems when childcare providers are verifying codes. You must not:

  • change the identity of the carer to that of a different person, for example by change all 4 data items to those of a different person
  • change the National Insurance number of the carer to one for different person
  • swap the carer and partner identity or details

If a child is moved to a different carer within the same local authority:

  1. The new carer must make a new application for childcare for working families.
  2. If the application is successful, you can create a new carer record and add the child to it.

It is not possible to transfer an eligibility code or child between carers. Any existing eligibility code for a child can continue to be used with a provider for as long as it is still valid for funding.

If the carer has died or is or no longer a foster carer and the partner becomes the carer, the partner must make a new application for childcare for working families. A code is always associated with a specific carer and child relationship. It is not possible to transfer an eligibility code from a carer to their partner or anyone else.

Change child details

Use the ‘Change’ link to edit a child’s details.

Making changes to child details after an eligibility code has been issued

After an eligibility code has been issued for a child, you can only make acceptable changes to the child details. For example, you can:

  • correct a mistake in the child date of birth or name including adding or removing middle names in the first name field. If you edit the child's date of birth after the issue or reconfirmation of a code, then the edited date of birth will not work if used to verify the eligibility code. Contact our service desk for help.
  • change the first name from a legal name to a preferred name, or the other way round
  • change the name of the child when their name legally changes

After the eligibility code has been issued, you cannot:

  • change the identity of the named child to that of a different child, for example by changing all 3 data items to those of a different child
  • swap the child details with those of another child looked after by this carer or another carer
