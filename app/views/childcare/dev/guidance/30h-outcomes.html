Childcare for working families eligibility checks: guidance on outcome pages

This guidance explains the validity outcomes, statuses and terminology used after a check has been carried out.

Term dates and eligibility code validity

The term dates for using eligibility codes are:

  • 1 January to 31 March – Spring term
  • 1 April to 31 August – Summer term
  • 1 September to 31 December – Autumn term

An eligibility code can be used in the next term after the eligibility confirmed on date. For example, if the eligibility confirmed on date is 21 February, then the code can be used in the summer term.

The eligibility confirmed on date is sometimes called the validity start date.

Parents can apply for 30 hours of free childcare from when their child is 23 weeks old. There is guidance for parents on when to apply and when they can get their hours, as this depends on the date the child turns 9 months old. Eligibility codes can be used the term after the child turns 9 months.

When you run a check, the outcome page tells you what term or terms a code is valid for.

Discretionary start dates

When a parent applies before the start of a term, but HMRC confirms their eligibility after that term has started, then the system sets a discretionary start date. This is based on the application submitted on date.

A discretionary start date is only applied when the eligibility confirmed on date is between 1st and 14th of the first month in the current term (January, April or September) and the application submitted on date is from the previous term. When the discretionary start date is applied, the code can be used in the current term.

For example, if the eligibility confirmed on date is 4 April (summer term) and the application submitted on date is 23 March (spring term), then a discretionary start date of 31 March (the last day of the spring term) is applied, and the code can be used in the summer term.

When you run a check, the outcome page will include a ‘discretionary start date applied’ tag to the ‘eligibility confirmed on’ field.

Grace periods

If a parent is no longer eligible for 30 hours of free childcare - either due to a change in circumstances or because they miss the reconfirmation date - a grace period applies if their child already has a childcare place. The child can continue in their current place for a short period, during which parents should submit another application or reconfirm their code.

Section A1.22 to A1.31 of the statutory guidance on GOV.UK provides detailed guidance for local authorities on grace periods. This includes guidance on notifying parents and grace period end dates.

When you run a check, the outcome page tells you whether a code is in the grace period and what the grace period end date is.

Temporary codes

Temporary codes are generated when there is an issue that prevents the parent from completing an application online, or there is a data issue that is incorrectly preventing them being made eligible. Temporary codes start with 11, can only be used once and cannot be reconfirmed.

A parent with a temporary code can apply for a permanent code at any time. If the pervious issues have been resolved, they will receive a permanent eligibility code (starting with 500 or 501). If there is another issue, they will receive another temporary code.

Temporary codes are not linked to permanent codes. HMRC do add a note to a parent's contact history when a temporary code has been issued.

It is possible for a temporary code to have overlapping validity dates with a permanent code. In this case, it's important to give the childcare provider the correct code.

Grace periods apply to temporary codes.

When you run a check, the outcome page tells you whether a code is a temporary code. If it is, the reconfirmation status will be 'Not applicable.'

Child age and compulsory school age

Parents can be eligible for 30 hours of free childcare if their child is aged 9 months to 4 years old. They can apply from when their child is 23 weeks old.

Child too young

Sections A.1.10 of the statutory guidance on GOV.UK provide information on the date when a child can take up their place.

When you run a check, the outcome page tells you if a child is too young to use their code or whether the code cannot be used until the following term.

Child too old

Parents are not eligible for 30 hours of free childcare when the child reaches compulsory school age. There is guidance on GOV.UK on when a child has reached compulsory school age.

When you a run a check, the outcome page tells you if the child has reached compulsory school age.
