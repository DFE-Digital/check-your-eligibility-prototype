Background

You can run a batch check to verify the eligibility of multiple children for free school meals.

Batch check guidance-steps

You have 3 steps to complete a batch check:

  • Download the CSV file template.
  • Fill in the template with the required information for each child.
  • Review any errors in the uploaded file and make any corrections before uploading.

Use a file size smaller than X MB.

Example of the batch check template

Upload the file

With the completed file you can upload it. Select the 'Run a batch check' on the dashboard.

Click on 'Choose file' (see below). You will be presented with a browsing window that allows you to find and select the file you created in the first step. Select the file and click 'Open'.

Run a batch check to process the file.

Errors

If there are issues with your data, you’ll see an error page listing the problem rows in the CSV file. For example: “Line 10: Issue with date of birth.” To fix this:​ Open the CSV file and locate the line mentioned (e.g., “Line 10”).​ Note: “Line 1” refers to the first row where you entered a parent’s information.​ Correct the data and save the file.​ Re-upload the updated file and click Run Check again.​

Results you may see

Entitled: The children of this parent or guardian are entitled to free school meals.​

May not be entitled: The children may not be entitled. Provide supporting evidence for your local authority to review.​

Could not check: The parent or guardian’s information does not match departmental records.​

Error: There was an issue accessing the data for this parent or guardian.
