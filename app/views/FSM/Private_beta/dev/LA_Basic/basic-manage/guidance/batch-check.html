You can run a batch check to verify the eligibility of multiple children for free school meals.

Batch check steps

You have 3 steps to complete a batch check.

  1. Download the CSV file template.
  2. Fill in the template with the required information for each child.
  3. Review any errors in the file and make any corrections before uploading.

Use a file size smaller than X MB.

Example of the batch check template

Upload the file

To upload the file, select Choose file.

You will be presented with a window where you can find and select the file you created and completed in the first step. Select the file, then select Open.

Select Run a batch check to process the file.

Errors

If there are issues with your data, you’ll see an error page listing the problem rows in the CSV file.

Example
Line 9: Issue with date of birth.

To fix the errors:

  • open the CSV file and locate the relevant line, for example 'Line 9'
  • Correct the data and save the file again
  • Upload the updated file and click Run a batch check

“Line 1” refers to the first row where you entered a parent’s information not row 1 on the spreadsheet.

Results of batch checks

Eligible targeted - The children of this parent or guardian are eligible for free school meals and additional support.​

Eligible expanded - The children of this parent or guardian are eligible for free school meals only.​

Not eligible at time of check - The children of this parent or guardian are not eligible for free school meals at the time of the check. If the parent or guardian thinks they are eligible, ask them to provide supporting evidence. View the list of acceptable evidence. ​

Information does not match records - We were unable to match the parent or guardian details with other government department records. Check that you entered the parent or guardian last name in full and that all other details are correct.​

System error - try again later - There was a problem with the service. You should try again later.
