Upload a CTF from your school management information system (MIS):

  1. Open your MIS (this could be SIMS or another system your school uses)
  2. Go to the admin area
  3. Select export pupil data
  4. Save the exported file on your computer
  5. Open the exported file and add any additional information that your system doesn't collect (for example, National Insurance numbers)
  6. Save the file again in XML format
  7. Upload the completed XML file below
